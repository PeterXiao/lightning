Performance

Maven plugin version 1.1.0 and Standalone JAR 5.2.0 benefit from significant performance optimisations. Both should process 500MB JMeter CSV file with 15.5 mln of records in under 90s, subject to memory limits. Overall performance of processing large CSV files should improve by up to 40% in comparison with previous versions.

To run Maven plugin with non-default 6GB memory limit:

mvn ... -DargLine="-Xmx6G"

To run Lightning standalone JAR with non-default 6GB memory limit:

java -jar -Xmx6G ...
